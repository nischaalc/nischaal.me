Shopaholic.

Shop Better.

In development for iOS. Currently in alpha.

Shopaholic is an app that wants to change the way you shop. No longer will you have to worry about finding something that you need - Shopaholic will tell you which stores have it, along with a whole ton of details about the item and how to get to the stores that are selling it! Add items to your Cart and once you feel like you've added enough, tell Shopaholic to do its' magic and it'll calculate which store is selling most, if not all, of the items that you want - for the lowest total price.

  • Built using Apple Xcode 5.
  • Designed in Adobe PhotoShop CS5.
  • Developed in collaboration with:
    • Zoltan Gercsak
    • Sean Swartz
    • Roger White

Paata!

Color everything!

Built for the web! Please contact me if you encounter any issues with the site!

As someone interested in design, I know that colors play a major role in the UX and picking the right ones can make or break your interface! Because of this I decided to build a color picker (this sort of thing) that would stand out from the rest. Paata (Sinhalese for "color") is the result of this experiment. The minimal interface presents three sliders that allow you to change RGB values and your changes are reflected in the change to the background. Paata gives you your color values in RGB, hex and generates a color that complements your color!
Built using HTML5, CSS3, JavaScript and jQuery. Feel free to contact me with suggestions or go ahead and fork the repository and submit a pull request when you're done making changes.
Kudos to:

  • Zurb for Foundation (provided under the MIT License), their amazing responsive framework!

Just Takes Five

Learn something.

Built for the web! Please contact me if you encounter any issues with the site!

Spare five minutes and learn something today, you never know when you'll need the knowledge.
Just Takes Five is a project aimed at helping you learn something new everyday. Pulling information from various websites, JT5 will present you with a random article from this carefully curated list of sources, ensuring that you get only the most quality content. So go ahead, learn something today and if you like what you've learned, come back tomorrow for more. It just takes five minutes.
Built using HTML5, CSS3, JavaScript and jQuery. Feel free to contact me with suggestions or go ahead and fork the repository and submit a pull request when you're done making changes.

Committed

A GitHub profile at a glance

In development.

Committed wants to help make your life a bit easier. Using the GitHub API, Committed collects information about a user of your choosing and displays all relevant information on a single page, in an easily readable form. Complete with information about repositories, users websites, popularity and committ history, Committed is ready to help.
Committed is an Open Source project, so feel free to fork the repo if you want to contribute.
